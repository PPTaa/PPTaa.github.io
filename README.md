# PPTaa의 주관적인 개발 블로그

안녕하세요? PPTaa의 지극히 주관적인 개발 블로그입니다.

Python 알고리즘 풀이와 Java Spring관련 자료들이 대부분입니다.

제가 학습했던 내용들을 주관적으로 작성해놨습니다~





## 관심있는 내용들

- Spring / Spring Boot
- AWS (Amazon Web Service)
- Docker (컨테이너, 이미지 배포)
- Jenkins (무중단 배포)
- MSA (Micro Service Architecture)
- Python 알고리즘 (코테 공부)
- 앱 개발 언어들 (Kotlin, Swift)
-  REST API







## 현재 업로드 중 내용...

- Python 알고리즘
- Spring 웹 어플리케이션 개발
